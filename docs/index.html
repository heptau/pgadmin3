# Quant a

pgAdmin III és la plataforma de codi obert més popular per a l'administració i el desenvolupament de PostgreSQL. El projecte original es va retirar a favor del pgAdmin 4 basat en web, però aquest fork comunitari manté viva l'aplicació nativa wxWidgets: corregeix errors per a versions modernes de PostgreSQL i afegeix desenes de funcions noves.

Aquesta versió presenta un port natiu complet per a macOS per a Apple Silicon (arm64), distribuït com un paquet .app autònom mitjançant Homebrew Cask.

  • License: Llicència PostgreSQL (permissiva, similar a BSD)
  • Repositori: github.com/heptau/pgadmin3
  • Fork ascendent: levinsv/pgadmin3 (Suport per a PostgreSQL 18)
  • Projecte original: postgres/pgadmin3 (archived)

# Instal·lació

macOS (Apple Silicon) — via més ràpida

Binari precompil·lat mitjançant Homebrew Cask:

brew install heptau/tap/pgadmin3

Inicia des de Launchpad, Spotlight o la terminal:

open /Applications/pgAdmin\ III.app

Compil·la des del codi font (macOS)

git clone https://github.com/heptau/pgadmin3.git
cd pgadmin3
make build
open "build-macos/pgAdmin III.app"

Nota: La primera compilació requereix una wxWidgets 3.3 compilada localment (amb --disable-std_containers). El bottle de Homebrew no és compatible — consulteu AGENTS.md per a més detalls.

git clone https://github.com/heptau/pgadmin3.git
cd pgadmin3
make build
./build/pgAdmin3

Windows

Obriu pgAdmin3.vcxproj al Visual Studio, o compileu creuat des de Linux amb mingw — consulteu INSTALL_EN.txt.

# About

pgAdmin III is the most popular open-source administration and development platform for PostgreSQL. The original project was retired in favor of the web-based pgAdmin 4, but this community fork keeps the native wxWidgets application alive — fixing bugs for modern PostgreSQL versions and adding dozens of new features.

This release introduces a fully native macOS port for Apple Silicon (arm64), distributed as a self-contained .app bundle via Homebrew Cask.

  • License: PostgreSQL License (permissive, BSD-like)
  • Repository: github.com/heptau/pgadmin3
  • Upstream fork: levinsv/pgadmin3 (PostgreSQL 18 support)
  • Original project: postgres/pgadmin3 (archived)

# Key Features

🖥️ Native macOS

A true native app built on wxWidgets 3.3 + Cocoa. No Electron, no Java — low memory footprint, fast startup.

🐘 PostgreSQL 12–18

Full support for modern PostgreSQL versions, PostgresPro Enterprise, Slony-I replication, and pgAgent scheduling.

🔍 SQL Editor

📊 Query Tool

Named bookmarks, join autocompletion via foreign keys, human-readable large numbers, cell-comparison diff view.

🔌 Plugins

Extensible via plug-in modules — add custom functionality without modifying the core.

🔐 SSL / SSH

SSL/TLS connection encryption and SSH tunnelling (libssh2) for secure server access.

# 概要

pgAdmin IIIは、PostgreSQL向けの最も人気のあるオープンソース管理・開発プラットフォームです。元のプロジェクトはWebベースのpgAdmin 4に移行して廃止されましたが、このコミュニティフォークがネイティブwxWidgetsアプリケーションを維持しています。最新のPostgreSQLバージョンへのバグ修正と、数多くの新機能を追加しています。

🔌 Додаци

# Инсталација

macOS (Apple Silicon) — најбржи пут

Унапред изграђени бинарни фајл путем Homebrew Cask:

brew install heptau/tap/pgadmin3

Покрените из Launchpad, Spotlight или терминала:

open /Applications/pgAdmin\ III.app

Изградња из изворног кода (macOS)

git clone https://github.com/heptau/pgadmin3.git
cd pgadmin3
make build
open "build-macos/pgAdmin III.app"

Напомена: Прва изградња захтева локално компајлирану wxWidgets 3.3 (са --disable-std_containers). Homebrew боца није компатибилна — погледајте AGENTS.md за детаље.

Linux

git clone https://github.com/heptau/pgadmin3.git
cd pgadmin3
make build
./build/pgAdmin3

Windows

Отворите pgAdmin3.vcxproj у Visual Studio-у или урадите унакрсно компајлирање из Linux-а са mingw-ом — погледајте INSTALL_EN.txt.

# 关于

pgAdmin III是最受欢迎的PostgreSQL开源管理和开发平台。原项目已停止开发，转而支持基于Web的pgAdmin 4，但这个社区分支延续了原生wxWidgets应用程序的生命——为现代PostgreSQL版本修复错误并添加了数十项新功能。

本版本引入了适用于Apple Silicon（arm64）的完全原生macOS移植版，通过Homebrew Cask作为自包含的.app捆绑包分发。

  • License: PostgreSQL许可证（宽松的BSD式许可证）
  • 仓库: github.com/heptau/pgadmin3
  • 上游分支: levinsv/pgadmin3 (支持PostgreSQL 18)
  • 原始项目: postgres/pgadmin3 (archived)

# 主要功能

🖥️ 原生macOS

基于wxWidgets 3.3 + Cocoa构建的真正原生应用。无需Electron，无需Java——内存占用低，启动速度快。

🐘 PostgreSQL 12–18

全面支持最新PostgreSQL版本、PostgresPro Enterprise、Slony-I复制以及pgAgent调度。

🔍 SQL编辑器

语法高亮、光标下语句自动识别、自动保存标签页、多结果窗口、PCRE文本转换。

📊 查询工具

命名书签、基于外键的JOIN自动补全、可读性优化的大数字显示、单元格比较差异视图。

🔄 Schema工具

跨服务器对象比较与HTML差异报告、基于GitLab的版本控制、通过amcheck进行B-tree索引检查。

📈 监控

服务器状态/活动窗口、进程高亮、等待事件采样（pg_wait_sampling）、AWR报告、CSV日志查看器。

🎨 SVG工具栏

现代化SVG工具栏图标，支持HiDPI/Retina，可针对不同服务器进行颜色编码。

🌐 本地化

内置11种语言，全部提供可编辑的.po源文件。捷克语、西班牙语、德语、法语、俄语、波兰语、中文和拉脱维亚语的缺失内容已在v2026.07.14中补全。

🔌 插件

可通过插件模块扩展——无需修改核心即可添加自定义功能。

🔐 SSL / SSH

SSL/TLS连接加密和SSH隧道（libssh2），确保服务器访问安全。
