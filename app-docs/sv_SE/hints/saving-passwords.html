Spara lösenord

VARNING: Du har valt att spara ditt lösenord. Det kommer att lagras i klartext i din hemkatalog på *nix-system, eller i din användarprofil på Windows. Om du inte vill att detta ska hända, tryck på knappen Avbryt.

pgAdmin använder PostgreSQLs 'pgpass'-mekanism för att lagra dina lösenord. På *nix-system kommer lösenordet att lagras i ~/.pgpass, medan det på Windows-system kommer att lagras i %APPDATA%\PostgreSQL\pgpass.conf (%APPDATA% är mappen 'Application Data' i din användarprofil). Denna mekanism används som standard av alla program som använder libpq-biblioteket för att komma åt servern, vilket inkluderar kommandoradsapplikationer som pg_dump och pg_restore, andra GUI-applikationer och drivrutiner som psqlODBC. Detta innebär att dessa applikationer automatiskt kan ansluta till servern med ditt sparade lösenord. Om du inte vill att detta ska hända bör du inte använda alternativet Spara lösenord i pgAdmin.
