Unicode-kodning på en Win32-server

Databasen är skapad för att använda Unicode (UTF8)-kodning för att lagra data och körs på en PostgreSQL 8.0-server som körs på en Win32-server.

PostgreSQL 8.0 Win32-implementeringen kan inte sortera teckensträngar enligt lämpliga sorteringssekvenser, eftersom operativsystemet inte erbjuder tillräckligt stöd för detta.

För denna installation rekommenderas det att använda en lokalspecifik kodning, köra klustret på ett annat operativsystem eller uppgradera ditt databaskluster till PostgreSQL 8.1, som har ytterligare kod för att hantera Unicode-sortering och lösa problemet.
