Salvataggio delle password

ATTENZIONE: Hai scelto di salvare la password. Verrà memorizzata in chiaro nella tua home directory sui sistemi *nix, o nel tuo profilo utente su Windows. Se non desideri che ciò accada, premi il pulsante Annulla.

pgAdmin utilizza il meccanismo 'pgpass' di PostgreSQL per memorizzare le password. Sui sistemi *nix, la password verrà memorizzata in ~/.pgpass, mentre sui sistemi Windows verrà memorizzata in %APPDATA%\PostgreSQL\pgpass.conf (%APPDATA% è la cartella 'Dati applicazioni' nel tuo profilo utente). Questo meccanismo è utilizzato per impostazione predefinita da tutti i programmi che utilizzano la libreria libpq per accedere al server, che include applicazioni a riga di comando come pg_dump e pg_restore, altre applicazioni GUI e driver come psqlODBC. Ciò significa che queste applicazioni possono connettersi automaticamente al server utilizzando la password memorizzata. Se non desideri che ciò accada, non dovresti utilizzare l'opzione Memorizza password in pgAdmin.
