Strumentazione del server

Al server mancano funzioni di strumentazione.

pgAdmin III utilizza alcune funzioni di supporto che non sono disponibili per impostazione predefinita in tutte le versioni di PostgreSQL. Queste consentono alcune attività che semplificano la gestione dei file di log e dei file di configurazione.

adminpack viene installato e attivato per impostazione predefinita se si utilizza l'installatore "un click" di PostgreSQL. Su Unix, potrebbe essere necessario installare il pacchetto contrib, tramite il proprio strumento di installazione dei pacchetti o tramite compilazione.

Una volta installata l'estensione, è sufficiente fare clic sul pulsante "Ripara!" affinché pgAdmin installi questa estensione nel database di manutenzione.
