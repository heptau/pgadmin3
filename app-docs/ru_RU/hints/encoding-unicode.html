Кодировка Unicode на сервере Win32

База данных создана с кодировкой Unicode (UTF8) для хранения данных и работает на PostgreSQL 8.0 на сервере Win32.

Реализация PostgreSQL 8.0 для Win32 не может сортировать строки символов в соответствии с правильными последовательностями сортировки, поскольку операционная система не предоставляет достаточной поддержки для этого.

Для этой установки рекомендуется использовать кодировку, специфичную для локали, запустить кластер на другой операционной системе или обновить кластер базы данных до PostgreSQL 8.1, который имеет дополнительный код для работы с сортировкой Unicode, решающий эту проблему.
