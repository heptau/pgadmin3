Редактирование представлений, хранимых процедур и функций

При редактировании кода представления, хранимой процедуры или функции, вместо использования маленького текстового поля в диалоге свойств объекта, возможно, удобнее использовать опции скриптов в pgAdmin для редактирования объекта в инструменте запросов. Это даёт преимущества редактора инструмента запросов, такие как автодополнение, поиск и замена, а также большее рабочее пространство.

Чтобы использовать опции скриптов, в окне браузера pgAdmin щёлкните правой кнопкой мыши по объекту, который хотите отредактировать, и выберите опцию Скрипт CREATE. Она может находиться в подменю Скрипты. Как вариант, вы можете выбрать объект, а затем выбрать опцию Скрипт CREATE из подменю Скрипты меню Инструменты.

Отредактируйте определение объекта по необходимости и используйте кнопку Выполнить запрос для применения изменений. Не забывайте, что изменение имени представления или функции, или сигнатуры функции или процедуры может привести к созданию нового объекта рядом с исходным. Этот полезный побочный эффект упрощает создание новых, но похожих объектов. Возможно, потребуется обновить браузер pgAdmin для отображения вновь созданных объектов.

Вы можете использовать эту технику для выполнения различных операций над разными объектами, однако синтаксис PostgreSQL CREATE OR REPLACE делает её особенно полезной для редактирования представлений, хранимых процедур и функций.
