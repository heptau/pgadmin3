Editar dados da tabela sem chave primária

Como a tabela não tem uma chave primária ou OIDs, você pode apenas visualizar os dados. Inserir novas linhas e alterar linhas existentes não é possível para a ferramenta Editar Dados sem chave primária.

Para editar dados, o pgAdmin III requer uma chave primária na tabela, o que é uma boa prática de design de banco de dados de qualquer forma. Alternativamente, a tabela pode ser criada WITH OIDS. Por favor, note que OIDs não são garantidos como únicos por um período muito longo de tempo, então usar OIDs como um tipo de chave primária é apenas uma segunda escolha.
