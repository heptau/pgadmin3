Enregistrer les mots de passe

ATTENTION : Vous avez choisi d'enregistrer votre mot de passe. Il sera stocké en texte clair dans votre répertoire personnel sur les systèmes *nix, ou dans votre profil utilisateur sous Windows. Si vous ne souhaitez pas que cela se produise, veuillez appuyer sur le bouton Annuler.

pgAdmin utilise le mécanisme 'pgpass' de PostgreSQL pour stocker vos mots de passe. Sur les systèmes *nix, le mot de passe sera stocké dans ~/.pgpass, tandis que sur les systèmes Windows, il sera stocké dans %APPDATA%\PostgreSQL\pgpass.conf (%APPDATA% est le dossier 'Application Data' de votre profil utilisateur). Ce mécanisme est utilisé par défaut par tous les programmes qui utilisent la bibliothèque libpq pour accéder au serveur, ce qui inclut les applications en ligne de commande telles que pg_dump et pg_restore, d'autres applications GUI, et des pilotes comme psqlODBC. Cela signifie que ces applications peuvent se connecter automatiquement au serveur en utilisant votre mot de passe stocké. Si vous ne souhaitez pas que cela se produise, vous ne devez pas utiliser l'option Enregistrer le mot de passe dans pgAdmin.
